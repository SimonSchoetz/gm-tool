# Random Tables

## Conclusion

The brainstorm below showed the concept "Random Tables" is complicated. There are random tables that help to quickly generate random content on the fly. Many of them are tied to specific source materials. It's not that straight forward to create general ones.
There are random tables that users might want to create, like for rolling on items -> that's source material specific.
There are random tables that are there to spark creativity, combining adjectives with nouns to create random names or descriptions.

It made clearer what type of lists there are. The idea for lists in adventure-bound context is that they are not thought as to be a list of items from source material but items important for the adventure, that need to be referenced in multiple sessions.
Other lists are probably source-material specific. And this is a different organizational problem on its own. Source Material would be some kind of a sibling to Adventures with its own set of lists.

Yet again, this opens up the question: How does one use the tool. Do you run many adventure with the same source material? Or do you run multiple adventures with different source materials each time?

For the GM-Tool: What do we care about? Minimalism is a big paradigm. And we care about the next session. Not how to organize the whole adventure at once.

## Usage

Let's say you are in the middle of a session and need to generate a NPC quickly. You want a good preconfigured NPC creator. You want the list of ancestries, maybe a tweak how common they are. I was just thinking how different ancestries have different types of skin or evern furr. We don't need to make a full fletched character builder like in Baldurs Gate 3. A type of ancestries already triggers imagination. So let's stick with very basic body types (tall to small, skinny to obese), obvious status (rich/powerfull to poor), age (young, middle-aged to old) and something that make them very specific like a "chipped horn" or something like that (thats something random I just came up with).
