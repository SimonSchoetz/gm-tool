# GM-Tool

Passion project to make my life as GM easier.

## Vision

Make GMing easy with an opinionated data framework and intuitive UI that guides through session preparation with tools like "Lazy DM" steps, quick random tables for names, loot, ect., encounter builder, session tracker, and many more.
